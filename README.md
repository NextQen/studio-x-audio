# 🎶 AudioBlend Studio

An all-in-one, browser-based audio processing tool built with the **Web Audio API**. Apply Slowed + Reverb, Studio EQ, and Noise Reduction simultaneously in a single click—no backend server required!

## 🚀 Features
- **One-Click Pipeline:** Apply multiple effects at once without multiple uploads.
- **Slowed & Reverb:** Dial in the perfect chill/lo-fi vibe.
- **Studio EQ:** Built-in bass booster to punch up the low end.
- **Noise Control:** Dynamics compressor acting as a basic noise gate.
- **100% Client-Side:** Fast processing right in your browser (No data uploaded to servers).

## 🛠️ Tech Stack
- HTML5 & Tailwind CSS
- Vanilla JavaScript (Web Audio API)
- Hosted on GitHub Pages
